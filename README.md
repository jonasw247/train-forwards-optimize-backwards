# Train Forwards, Optimize Backwards: Neural Surrogates for Personalized Medical Simulations

<img width="1966" height="820" alt="image" src="https://github.com/user-attachments/assets/23df9af5-042f-475e-9b61-d964977cbfa1" />
A differentiable neural surrogate model is trained on the forward tumor simulation
problem. It inputs the PDE coefficients and the brain tissue and outputs a tumor concentration.
Finally, the patient-specific coefficients are estimated using gradient-based optimization.
